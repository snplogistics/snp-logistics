# SNP Logistics LLC - Static Website

Simple static site scaffold for SNP Logistics LLC. Uses plain HTML/CSS/JS and small SVG icons.

Quick start:

1. Install dependencies (none required).
2. Serve locally:

```bash
cd snp-logistics
python3 -m http.server 8000
# Open http://localhost:8000
```
